Comprehensive RCT Search

Multi-source NCT ID discovery combining PubMed, Europe PMC, and ClinicalTrials.gov for maximum recall.

How it works: This search combines 4 data sources to find NCT IDs that might be missed by a single-source search:
  1. PubMed Articles: Extracts NCT IDs from abstracts, AccessionNumbers, and ArticleIds
  2. Europe PMC: Searches for NCT IDs in full-text articles
  3. PubMed-CT.gov Linkage: Uses NCBI's direct database links
  4. CT.gov API: Direct condition search
Note: This comprehensive search improves recall from ~88% (CT.gov API alone) to higher levels by cross-referencing published literature. Results depend on how often trials cite their NCT IDs in publications.

Comprehensive RCT Search

The Comprehensive RCT tab combines 4 data sources to maximize recall:

  1. PubMed Articles: Extracts NCT IDs mentioned in abstracts, AccessionNumbers, and ArticleIds
  2. Europe PMC: Searches European literature for NCT ID mentions
  3. PubMed-CT.gov Linkage: Uses NCBI's elink to find direct database connections
  4. CT.gov API: Direct condition-based search

This approach can improve recall from ~88% (CT.gov API alone) to higher levels for conditions where trials have published results.
