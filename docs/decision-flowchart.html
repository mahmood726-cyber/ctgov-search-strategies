Quick Recommendations

  • Systematic Reviews: Use S1 (maximum recall)
  • RCT Meta-analysis: Use S3 (best balance)
  • Rapid Reviews: Use S7 or S8
  • Treatment Focus: Use S10
  • Never rely on a single search source
  • Always search WHO ICTRP too
