Interpretation: This forest plot shows CT.gov search recall stratified by therapeutic area. Point estimates represent the proportion of gold standard trials retrieved using the combined search strategy (S4). Horizontal lines indicate 95% Wilson score confidence intervals. The vertical reference line at 75% represents the overall recall target. Areas above the reference line (Respiratory, Diabetes, Rheumatology) achieve higher-than-average recall. Oncology shows notably lower recall (65%) due to combination therapy challenges.
